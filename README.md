# Golang-Compartamos
Actividad Tecnica de Golang - Firebase
## Install Mux library

```bash
go get -u github.com/gorilla/mux
cloud.google.com/go/firestore
google.golang.org/api/option
google.golang.org/api/iterator
```

## Build

```bash
go mod init compartamos-clientes
go build
```

## Run

```bash
go run .
```
